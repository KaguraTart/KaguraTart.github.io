使用hugo在Github上自动部署博客

August 20, 2022

本文概要：

  • hugo安装和部署
  • GitHub的repository创建
  • GitHub的action创建

0x00 Linux(ubuntu)下hugo的下载和安装

使用apt进行二进制安装

$ sudo apt-get hugo

0x01 使用hugo生成站点

使用Hugo快速生成站点

$ hugo new site <your project filename>

举例子：比如希望生成到 /myweb/tart 路径文件夹：

$ hugo new site /myweb/tart

0x02 创建文章

方法1

使用hugo命令创建：

$ hugo new about.md

about.md 自动生成到了 content/about.md ，打开 about.md 看下：

---
date = "2015-10-25T08:36:54-07:00"
draft = true
title = "about"
---
正文内容

0x04 安装主题

到！主题 挑选一个心仪的皮肤，比如你觉得 Hyde 皮肤不错，找到相关的 GitHub 地址，创建目录 themes，在 themes 目录里把皮肤 git clone 下来：

# 创建 themes 目录
$ cd quickstart
$ git init
$ git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke

0x05 运行hugo

使用命令行运行hugo并部署

$ hugo server -D
